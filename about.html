Sobre mim

Sou um profissional em transição para a área de ciência de dados, estudando programação com foco em análise e modelagem de dados. Tenho experiência com lógica de programação, adquirida ao longo de um ano de estudos para concursos, e um forte domínio do Excel avançado, o que me permite lidar com grandes volumes de dados de forma estruturada. Atualmente, estou aprofundando meus conhecimentos em Python e Machine Learning, além de explorar fundamentos de desenvolvimento web com JavaScript, HTML e CSS como parte do meu curso de ciência de dados.

Meu objetivo é aplicar essas habilidades para extrair insights valiosos a partir de dados, utilizando técnicas estatísticas, aprendizado de máquina e visualização de informações. Com um olhar analítico e capacidade de resolver problemas, estou me preparando para atuar profissionalmente na área, desenvolvendo soluções baseadas em dados para otimizar processos e apoiar a tomada de decisões estratégicas.
